# Имитационное обучение в виртуальной среде
Система состоит из 2х компонентов: игра и нейронные вычисления. В игре реализована простая логика, клеточное пространство, содержащее препятствия, ключи и награды. В нейронных вычислениях реализована математика нейронных слоев. Сама программа объединяет компоненты в нескольких режимах, которые запускаются с помощью терминала.
